# 10 Browser Privacy Settings You Should Change Right Now

## Why Default Browser Settings Are Privacy-Hostile

Chrome is made by Google — the world's largest advertising company. The default settings reflect this relationship, favouring data collection over user privacy.

## 1. Block Third-Party Cookies

**Chrome:** Settings → Privacy and Security → Cookies → Block third-party cookies  
**Firefox:** Settings → Privacy & Security → Enhanced Tracking Protection → Strict

## 2. Disable Telemetry

**Chrome:** Settings → You and Google → Sync and Google services → disable usage statistics  
**Firefox:** Settings → Privacy & Security → Firefox Data Collection → uncheck all

## 3. Enable DNS-over-HTTPS

**Chrome:** Settings → Privacy and Security → Security → Use secure DNS → Cloudflare  
**Firefox:** Settings → Privacy & Security → DNS over HTTPS → Max Protection

## 4. Turn Off Autofill for Addresses and Cards

Payment data stored in browser profiles syncs to cloud servers. Use a local password manager instead.

## 5. Disable Page Prefetching

Chrome: Settings → Privacy and Security → disable "Preload pages for faster browsing". This feature fetches pages you haven't visited, triggering tracking pixels on sites you never chose to open.

## 6. Review Site Permissions

Settings → Site Settings — audit which sites have access to location, camera, microphone, and notifications. Revoke anything not actively needed.

## 7. Disable WebRTC (Firefox)

In `about:config`, set `media.peerconnection.enabled` to `false` to prevent WebRTC from exposing your real IP even through a VPN.

## 8. Use HTTPS-Only Mode

Firefox: Settings → Privacy → HTTPS-Only Mode → Enable in all windows.

## 9. Disable Search Suggestions

Search suggestions are sent to the search engine as you type — before you press Enter.
