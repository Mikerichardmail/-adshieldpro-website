# How to Enable Strict Mode in AdShield Pro for Maximum Blocking

## What Strict Mode Does

AdShield Pro's Strict Mode enables an extended set of filter lists beyond the default EasyList + EasyPrivacy combination, providing more comprehensive blocking at the cost of a slightly higher false positive rate.

## Enabling Strict Mode

1. Click the AdShield Pro shield icon
2. Navigate to **Settings → Filtering Level**
3. Select **Strict**
4. Confirm the prompt

## Additional Lists Enabled in Strict Mode

- **uBlock Filters — Annoyances** — cookie consent banners, newsletter popups, chat widgets
- **Peter Lowe's Ad and Tracking Server List** — expanded ad server coverage
- **Malware Domain List** — known malicious domains
- **Fanboy's Social Blocking List** — social media tracking widgets
- **Regional EasyList variants** — Germany, France, Spain, Russia, and others based on your browser language

## What Gets Blocked That Standard Mode Misses

- Cookie consent popups (auto-dismissed or blocked outright)
- Newsletter signup overlays
- Live chat widgets from third-party services
- Social media sharing buttons (which act as tracking pixels)
- Additional ad networks primarily serving non-English markets

## Handling Breakages in Strict Mode

Strict Mode has a higher false positive rate than Standard mode. If a site breaks:

1. Click the AdShield Pro icon
2. Select **"Use Standard mode for this site"**
3. This drops back to Standard mode for that specific domain while keeping Strict active everywhere else

Report persistent false positives via the [support page](https://adshieldpro.org/support) so they can be fixed in filter list updates.
