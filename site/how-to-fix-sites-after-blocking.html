How to Fix Broken Sites After Blocking

If a website doesn’t load correctly after blocking ads/trackers, it usually depends on a third-party script. The fix is typically a quick per-site adjustment.

1) Reload and check the extension state

Reload the page first. Then confirm AdShield Pro is enabled for the current tab and the site isn’t allowlisted.

2) Disable blocking for that site

If the site relies on blocked scripts, temporarily disable blocking for that site and refresh. This is the fastest way to confirm the cause.

3) Use allowlisting when needed

If you trust the site and it keeps breaking, add it to your allowlist so it always works. You can remove it later.

4) Still stuck? Get help

Some sites require more tuning. Use the support page for next steps and contact details.
